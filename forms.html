Forms

There are two ways to add forms to your app:

  1. With a shortcode in the WordPress content editor.
  2. With a function in a PHP template.
  Shortcode PHP Function
Login Form [loginform] <?php echo wpwebapp_login(); ?>
Sign-Up Form [signupform] <?php echo wpwebapp_signupform(); ?>
Password Reset [pwreset] <?php echo wpwebapp_pwreset(); ?>
Password Change [pwchangeform] <?php echo wpwebapp_pwform(); ?>
