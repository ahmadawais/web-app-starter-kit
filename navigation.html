User-Specific Navigation

You may want to show different navigation elements to users who are logged in versus those who are not. Simply include the following code in your header.php file:

<?php if (is_user_logged_in()) : ?>
    <!-- Logged In Navigation -->
<?php else : ?>
    <!-- Logged Out Navigation -->
<?php endif; ?>

To display the current user's username, use this script:

<?php $current_user = wp_get_current_user(); echo $current_user->user_login; ?>
